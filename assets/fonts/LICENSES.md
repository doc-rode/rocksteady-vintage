## Notes
- These `woff2` files are the **latin / latin-ext subsets** as delivered by the Google Fonts
  API (extracted from the self-contained design preview). They are unmodified OFL fonts and
  fully cover the English content on this site.
- The OFL requires the license to accompany the font files. For the cleanest provenance,
  download the full families from the source repositories above (or the Google Fonts
  "Download family" button) and drop their upstream `OFL.txt` into this folder — the exact
  copyright lines and any Reserved Font Names live there.
- Do not sell the font files on their own, and do not release a modified font under the
  original family name (OFL Reserved Font Name clause). This site does neither.
